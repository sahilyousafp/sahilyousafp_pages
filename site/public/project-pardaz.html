Pardaz Exhibition Simulation

A production-grade pedestrian simulation platform for validating exhibition layouts. Architects draw a floor plan, configure visitor flows, and the backend runs multi-agent simulation — returning heatmaps, centrality scores, and crowding predictions.

Overview+

Validate exhibition spaces before you build them

Exhibition designers traditionally rely on intuition and post-occupancy data to understand visitor flow. Pardaz gives them a predictive simulation tool that answers critical questions at the design stage: Where will crowds form? Which paths will be underused? Are there bottlenecks near key exhibits?

The architect draws directly on an uploaded floor plan image using the annotation canvas, marking pedestrian entry points, exit gates, exhibit locations, and barriers. The backend converts this annotation into a spatial graph and runs agent-based pedestrian simulation.

Results arrive as interactive heatmap overlays and centrality scores — the same metrics used in urban planning, now applied to interior exhibition design.

User Flow+

Two-page workflow

01

Upload Floor Plan

Upload an image of any exhibition floor plan. The canvas renders it as a live drawing surface.

02

Annotate

Use the Toolbar to mark entrances, exits, barriers, exhibits, and pedestrian zones directly on the plan.

03

Run Simulation

The backend runs multi-agent pedestrian simulation on the spatial graph derived from your annotations.

04

Analyse Results

Heatmap overlay shows density. Centrality Panel ranks nodes by betweenness — pinpointing bottlenecks.
