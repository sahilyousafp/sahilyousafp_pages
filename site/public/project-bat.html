Building Analysis Tool

An AI-driven accessibility tool that analyses healthcare building layouts for visually impaired navigation. Combining graph-based spatial analysis, image segmentation, and RAG-powered design recommendations — published at ANNSIM 2025.

Overview +

AI-driven pathfinding for hospitals

The Building Analysis Tool (B.A.T.) was designed to address a critical gap in healthcare architecture: the lack of AI-powered tools to assess and optimise accessibility for visually impaired users.

The tool ingests IFC or 3D building models and produces: graph-based navigation analysis to identify bottlenecks; semantic image segmentation of interior photography to detect key accessibility elements; and RAG-powered recommendations that query a curated knowledge base of accessible design standards.

A priority scoring framework ranks each space (1–10) by its criticality for independent navigation — from entrance and reception (highest) to staff-only storage (lowest).

Published Research ANNSIM Conference 2025 · Society of Modelling & Simulation International (SCS)
Pipeline +

How it works

01

IFC Model Import

Architect uploads IFC or 3D model; parser extracts rooms, corridors, doors, and ramps into a spatial graph.

02

Graph Analysis

NetworkX builds a weighted navigation graph. Betweenness centrality and shortest-path algorithms identify bottlenecks.

03

Image Segmentation

Interior photographs are segmented to highlight tactile surfaces, ramps, signage, and door widths.

04

RAG Recommendations

An LLM queries an accessibility standards knowledge base and outputs tailored design improvement suggestions.

Data +

Space priority scoring

A key research contribution was developing a priority hierarchy for healthcare spaces, used to weight the graph analysis and direct AI recommendations toward the most critical navigation zones.

Space TypePriorityCriticality
Entrance / Main Access10 / 10Critical — first point of independent navigation
Reception / Information9 / 10Essential for orientation and wayfinding
Restrooms9 / 10High-frequency need, complex spatial layout
Examination Rooms8 / 10Destination spaces, require precise navigation
Corridors / Lifts8 / 10Key connective tissue of movement
Waiting Areas6 / 10Social spaces with ambient complexity
Staff Areas3 / 10Occasional access, minimal patient use
Storage / Utility1 / 10Non-patient-facing, lowest priority
