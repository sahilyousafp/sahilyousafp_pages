Overview

Making land change visible to everyone

The AI4ALL project asks a fundamental question: how do you include citizens in decisions about their environment when the data that drives those decisions is locked behind technical interfaces?

This platform answers by turning OpenLandMap satellite data into an interactive split-screen comparison — anyone can drag a divider to reveal how land cover has changed between any two years from 2000 to 2023. The underlying data is real satellite imagery at 250m resolution, globally.

Two modes serve two audiences: Data mode for analysts (raw land-cover tiles, opacity controls, quantified legend) and Story mode for citizens (narrative overlays that contextualise what the map is showing). A MorphLayer and LoupeLens enable localised magnification for community discussion.

Interface Modes

Data mode vs Story mode

Raw land-cover tiles. Year comparison slider. Opacity control. Quantified legend with land class breakdown.

Contextual information panel explaining the participatory decision-making thesis underlying the visualisation.

Architecture

Component structure

  App.tsx
  ├── Leaflet Map  (#map div)
  │   ├── CartoDB basemap
  │   ├── leftLayer  (year A tile layer)
  │   └── rightLayer (year B tile layer)
  │
  ├── IdeologyPanel   — participatory context
  ├── LossLayer       — land loss highlighting
  ├── MorphLayer      — story mode overlays
  └── LoupeLens       — magnification lens

  Data mode:
  ├── Split slider (L.control.sideBySide)
  ├── Year chip labels (left / right)
  ├── Opacity range control
  └── Legend panel (hover → popup)

  Story mode:
  ├── IGN aerial imagery replaces tile layers
  ├── LoupeLens active
  └── Data UI hidden
