Overview

A city that thinks

Traditional agent-based models use fixed rule tables for movement. This project replaces those rules with a live LLM: each step, the agent reads its own memory, perceives its surroundings, and reasons in natural language about where to go next.

Built on the Eixample district of Barcelona — one of the world's most regular urban grids — the simulation uses real street network data from Overture Maps and OpenStreetMap, with full spatial querying via DuckDB.

Each agent has a persistent KV Memory (structured current state) and Stream Memory (append-only event log), making every trajectory a readable story of urban behaviour.

Agent Archetypes

Four urban personas

Each agent is assigned an archetype that shapes its LLM prompt, driving realistic and demographically varied movement patterns across the city grid.
